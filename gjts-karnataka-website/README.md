# Government Junior Technical Schools Karnataka - Website

A premium website for the six Government Junior Technical Schools across Karnataka, featuring modern UI/UX, AI-powered chatbot, and alumni registration system.

## Six GJTS Locations
1. GJTS Ballari
2. GJTS Bhadravati
3. GJTS Hubballi
4. GJTS Bagalkot
5. GJTS Kalburgi
6. GJTS Mangalore

## Features
- Modern, responsive design with premium UI/UX
- AI-powered chatbot trained on school data
- Individual school pages with detailed information
- Alumni registration and management system
- Student portal
- Admission information and online application
- News and events section
- Gallery and achievements showcase

## Tech Stack
- Frontend: React.js with Next.js
- Styling: Tailwind CSS + Framer Motion
- Chatbot: OpenAI API / Custom trained model
- Backend: Node.js + Express
- Database: MongoDB
- Authentication: JWT
