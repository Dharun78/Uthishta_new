# School Images

This folder contains images for the 6 GJTS schools.

## Current Images

The system uses placeholder images from Unsplash for now. You can replace these with actual school photos.

### Image Specifications:
- **Size**: 800x600px (4:3 aspect ratio)
- **Format**: JPG or PNG
- **Quality**: High resolution for web

### School Images Needed:
1. `ballari.jpg` - GJTS Ballari campus
2. `bhadravati.jpg` - GJTS Bhadravati campus
3. `hubballi.jpg` - GJTS Hubballi campus
4. `bagalkot.jpg` - GJTS Bagalkot campus
5. `kalburgi.jpg` - GJTS Kalburgi campus
6. `mangalore.jpg` - GJTS Mangalore campus

### How to Add Real Images:
1. Take or obtain high-quality photos of each school
2. Resize to 800x600px
3. Save with the exact filenames above
4. Place in this folder (`public/images/`)
5. Refresh the website

### Temporary Solution:
The SchoolCard component will use placeholder images from Unsplash if local images are not found.
